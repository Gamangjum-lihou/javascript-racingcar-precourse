# 자동차 게임 문서 작성

## 기능 체크리스트

- [ ] 자동차 이름 입력

  - [ ] 5자 이하
  - [ ] 콤마로 구분된 이름

- [ ] 시도 횟수 입력

  - [ ] 숫자 확인

- [ ] 게임 시스템 실행
  - [ ] 0 ~ 9까지의 무작위 값을 구한 뒤, 무작위 값이 4 이상일 경우 전진 이외의 경우에는 멈추는 기능을 실행한다.
  - [ ] 위의 기능을 입력받은 시도 횟수만큼 반복한다.
  - [ ] 시도 횟수 만큼 반복했을 경우, 우승자를 구한다.
  - [ ] 우승자가 여러 명일 경우 쉼표를 이용하여 구분한다.
